# AI Video Studio Pro - توثيق إضافي

## بنية المجلدات
- **app.py**: التطبيق الرئيسي (Streamlit)
- **web_assets/**: ملفات الواجهة الأمامية (HTML, CSS, JS)
- **audio_library/**: مكتبة المؤثرات الصوتية والموسيقى
- **videos/input/**: مجلد رفع الفيديوهات
- **videos/output/**: مجلد النتائج
- **data/**: بيانات مؤقتة أو إعدادات

## طريقة التشغيل
- شغّل `setup.bat` أولاً (مرة واحدة)
- ثم `run.bat` أو `streamlit run app.py`

## الدعم
لأي استفسار أو تطوير إضافي، تواصل مع المطور.
